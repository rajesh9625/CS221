<a NAME = "research"></a>
<h2>Research</h2>

<P>My research interests are in the fields of databases and information
retrieval, including search, data-intensive computing,
data integration and sharing, data warehouses, data cleansing,
and Web information management.  The following are several database
projects I am working on. <BR>

<h3>Current Projects</h3>
<ul>
<li> <a href="http://asterix.ics.uci.edu/">ASTERIX</a>: This project is about
Active, Scalable, Transactional Enterprise Repository for Information in XML. It's a new effort to develop a scalable semi-structured information management system, based on XML and XQuery technologies, targeting very large shared-nothing compute clusters.

  </ul>

<h3>Past Projects</h3>
<ul>
<li><a href="http://flamingo.ics.uci.edu/">FLAMINGO</a>: A project on data cleaning. 

<li><a href="http://ipubmed.ics.uci.edu/">IPUBMED</a>: Efficient instant
  search on large amounts of data.  It started with the joint
  research <a href="http://tastier.ics.uci.edu/">project</a>
  with <a href="http://www.tsinghua.edu.cn">Tsinghua University</a> on
  efficient auto-complete and type-ahead search on large data sets.

<li><a href="http://fr.ics.uci.edu/">Family Reunification</a>.
Help people find their loved ones during or after a disaster.
<li><a href="http://www.ics.uci.edu/~raccoon/">The Raccoon Project on Data
Integration and Sharing</a>.  I started this project several years ago,
and it's in its final stage. I still have some ongoing research related to
this project. But compared to the first two projects, this one is less active.
<li> <a href="http://www.ics.uci.edu/~chenli/webobjects/">Data sets of the history 
of data objects collected from 6 web sites in 1.5 years.</a>. 
</ul>

  <li> <a href="http://www.ics.uci.edu/~salsubai/rstartree.html">

  <li> <a href="http://jujube.ics.uci.edu/www/learning.html">Instant fuzzy

  <li> <a href="http://www.omniplaces.com/">Location-based instant fuzzy
  search</a>.

  <li> <a href="http://jujube.ics.uci.edu/localsearch/fuzzysearch/">Location-based approximate keyword search</a>.

  <li><a href="http://chime.ics.uci.edu/">CHIME<a>: Error-tolerant Chinese

  <li> <a href="http://asterix.ics.uci.edu/fuzzyjoin/">Efficient Parallel
  Set-Similarity Joins Using MapReduce</a>.

  <li> <a href="http://fr.ics.uci.edu/haiti/">Haiti family
